Como usar esta ferramenta:

1. Obtenha uma chave API gratuita em NewsAPI.org

2. Cole a chave no campo de configuração e salve

3. Selecione uma categoria de notícias OU digite uma palavra-chave

4. Copie a URL do feed gerado

5. Use esta URL em suas automações ou aplicativos

6. O feed será atualizado automaticamente com as últimas notícias

Hospedagem Gratuita:

Para hospedar este gerador de feeds gratuitamente e tê-lo disponível 24/7:

  1. Faça uma conta no GitHub (gratuita)
  2. Salve este código em um arquivo chamado "index.html"
  3. Crie um novo repositório no GitHub
  4. Faça upload do arquivo para o repositório
  5. Ative o GitHub Pages nas configurações do repositório
  6. Seu feed estará disponível em: https://seuusuario.github.io/nomedorepositorio

Exemplo de URL para automação: https://seuusuario.github.io/nomedorepositorio/?category=technology&format=json

Exemplo de URL com palavra-chave: https://seuusuario.github.io/nomedorepositorio/?keyword=inteligência+artificial&format=json
